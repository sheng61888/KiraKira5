Matrices & Operations

Recognise matrix order, test equality, and carry out valid addition, subtraction, and multiplication so you can model transformations and systems exactly the way SPM questions expect.

Lesson Overview

A matrix is simply a rectangular array with dimensions rows x columns. Two matrices are equal only when their orders and every corresponding entry match. Addition and subtraction are performed entry-wise, while multiplication AB is possible only when the number of columns in A equals the number of rows in B, resulting in a matrix whose order is determined by the outer dimensions.

Thinking about rows and columns as vectors helps when performing dot products for multiplication. You will also read contexts such as transformation matrices or tabular data, so labelling your matrices clearly prevents order mistakes.

Operations to Master

  • State the order of every matrix before deciding what operation is possible.
  • Use element-by-element addition or subtraction only when orders match exactly.
  • For multiplication, write the row of the first matrix and column of the second matrix you are combining to avoid mixing indices.
