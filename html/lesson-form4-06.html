Linear Inequalities in Two Variables

Sketch boundary lines, shade half-planes, and locate overlapping regions for systems so you can read solutions straight from the graph.

Back to Catalogue

Lesson Overview

Inequalities such as ax + by <= c describe half-planes. The boundary line ax + by = c is solid for <= or >= and dashed for < or >. Pick a test point (often (0,0)) to decide which side to shade.

Systems combine several inequalities. Their solution set is the overlapping region. Many optimisation problems restrict x and y to the first quadrant (x >= 0, y >= 0), so be comfortable interpreting these geometric constraints.

Key Skills

  • Convert inequalities to boundary-line form and determine intercepts.
  • Decide whether the boundary is solid or dashed and test shading.
  • Graph feasible regions for systems and interpret coordinate solutions.
