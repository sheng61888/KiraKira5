Number Bases & Conversions

Learn how digits, place value, and conversion methods change across bases so you can confidently switch between binary, octal, decimal, and hexadecimal representations.

Back to Catalogue

Lesson Overview

A number base tells you which digits are allowed and how each place represents powers of that base. In any base b (where b is at least 2) the valid digits run from 0 up to b minus 1. That means base 2 allows only 0 and 1, base 8 allows 0 through 7, base 10 allows 0 through 9, and base 16 extends 0 through 9 with the letters A to F for values 10 to 15.

A numeral such as (dk dk-1 ... d1 d0)b represents the expanded sum dkbk + dk-1bk-1 + ... + d1b + d0. The structure mirrors the familiar decimal system; only the allowed digits and the base of each power change.

Source: Pandai

Conversions You Must Know

  • Base b to base 10: Expand using place value and sum digit times b to the relevant power.
  • Base 10 to base b: Use repeated division by b and read the remainders from last to first.
  • Between non-decimal bases: Convert via base 10 or group digits when bases are related (for example binary to octal or hex).
