Probability of Combined Events

Combine events with unions, intersections, independence, and conditional reasoning to tackle multi-step experiments confidently.

Lesson Overview

For any events A and B, use P(A U B) = P(A) + P(B) - P(A n B). Mutually exclusive events have P(A n B) = 0. Independent events satisfy P(A n B) = P(A)P(B) or P(A | B) = P(A). Without replacement typically means dependence.

Tree diagrams and tables help visualise sequential events. Conditional probability P(A | B) = P(A n B) / P(B) keeps your logic precise.

Key Skills

  • Decide when events are mutually exclusive, independent, or dependent.
  • Apply addition and multiplication rules accurately.
  • Use conditional probability to interpret real contexts (with or without replacement).
