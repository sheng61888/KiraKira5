Congruent, Enlargement & Combined Transformations

Review rigid motions, scale figures up or down (including negative scale factors), and chain multiple transformations to describe tessellations and exam diagrams confidently.

Back to Catalogue

Lesson Overview

Congruent figures share the same shape and size and can be mapped using translations, rotations, or reflections. Enlargement introduces a scale factor (k) that multiplies lengths by (k) and areas by (k^2). Negative scale factors reflect through the centre before enlarging. Combined transformations require you to describe the actions in order, while tessellations rely on repeating shapes without gaps.

We group these skills under the same chapter so you can connect congruency tests, vector descriptions of translations, rotation details (centre, angle, direction), and enlargement coordinates in one go.

Things to Emphasise

  • State the centre and scale factor whenever you describe an enlargement; mention sign changes if k is negative.
  • For combined moves, write them in the order performed because reflect-then-rotate differs from rotate-then-reflect.
  • Remember that a regular tessellation needs one regular polygon repeated with no overlaps.
