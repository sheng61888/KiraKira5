Form 4 - Module 03

Logical Reasoning: Statements & Arguments

Treat logic as math for sentences. Learn how statements combine, how arguments are built, and how to spot fallacies before they trap you.

Back to Catalogue

Lesson Overview

A statement is any sentence that can be labelled true or false. We build complex statements with AND (written as and or P && Q), OR (P or Q), and NOT (not P), then track their truth values using tables. Conditional statements (if P then Q) have relatives: converse, inverse, and contrapositive.

Arguments chain statements together. An argument is valid when true premises force the conclusion to be true. Validity depends on structure, not the topic of the sentence. Classic traps include affirming the consequent and denying the antecedent.

Key Skills

  • Classify sentences as statements or non-statements.
  • Write truth tables and find negations, converses, and contrapositives.
  • Test arguments for validity and identify common fallacies.
